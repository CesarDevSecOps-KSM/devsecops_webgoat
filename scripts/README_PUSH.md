# Como subir usando GitHub Desktop

1. Abra GitHub Desktop e selecione o repositório local correspondente (Repository -> Show in Explorer).
2. Copie todo o conteúdo desta pasta (o projeto) para a pasta do repositório local que o GitHub Desktop abriu.
3. Volte ao GitHub Desktop, escreva um summary no campo de commit (ex: 'Add project files') e clique em Commit to main.
4. Clique em Push origin para enviar ao GitHub.
